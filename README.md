# 💻STM32

这里是一些学习STM32过程中的笔记和代码。你可以通过点击下面笔记目录中的链接在线阅读这些笔记，也可以从GitHub仓库中下载markdown格式的笔记。

## 📝Notes and code

- STM32的每一章都存储在一个文件夹中，其名称以数字开头。在这些文件夹中：

    - 笔记以markdwom文档格式提供。
    - code文件夹用来存储实现学习内容的代码
    - 在大多数情况下，笔记和代码都是基于STM32F103C8T6开发板，基于HAL库编程的，开发板原理图已经上传。
